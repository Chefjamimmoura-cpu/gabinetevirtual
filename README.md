# Gabinete Virtual - Carol Dantas

Este é o repositório frontend do sistema Gabinete Virtual, construído com Next.js (App Router).

## Regras e Informações Críticas do Projeto

- **Usuária Principal:** Cynthia
- **Email de Contato:** cynthiasouzap@gmail.com
- **Direitos Autorais:** Todos os Direitos Reservados a Jamim Moura Santos.
- **Desenvolvimento:** O projeto é desenvolvido cooperativamente pelas IAs Antigravity (Frontend e Lógicas Leves) e Claude Code (Backend, Infra e Lógicas Pesadas).

## Arquitetura

- **Frontend:** Next.js 15+
- **Estilização:** CSS Vanilla com Glassmorphism
- **Banco de Dados/Auth:** Supabase
- **Serviços Secundários:** O sistema integra-se com o `cmbv-parecer` (Node.js/Puppeteer).

## Scripts Disponíveis

```bash
npm run dev
# Inicie o servidor de desenvolvimento em http://localhost:3000
```
